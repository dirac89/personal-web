# Personal Portfolio – Javier Aguilera

This is my personal portfolio built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**. It showcases my work as a Senior Data Scientist, including selected projects, experience timeline, and contact information.

## 🚀 Tech Stack

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [React Icons](https://react-icons.github.io/react-icons/)

## 📁 Project Structure

```
src/
├── assets/          # Static files (profile picture, logos, etc.)
├── components/      # React components (Hero, Projects, About, etc.)
├── data/            # JSON-style data (projects, timeline)
├── hooks/           # Custom React hooks
└── App.jsx          # Main layout
```

## 🚧 Development

To run the project locally:

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

## 🚀 Deployment with Vercel

This project is deployed using [Vercel](https://vercel.com).

### One-time setup

1. Push your code to a GitHub repository
2. Go to [vercel.com](https://vercel.com), log in and import the repo
3. Use the following settings:
   - **Framework preset:** Vite
   - **Build command:** `npm run build`
   - **Output directory:** `dist`
4. Click **Deploy**

### Continuous Deployment

Every push to `main` (or your configured branch) will automatically trigger a redeploy.
